Deployment

Gluu Flex is cloud agnostic--use Amazon, Google, Microsoft or build your own cloud!

Self Host

Gluu Flex is our commercial distribution of the Janssen Project that includes extra business features like an admin web interface. Enterprise Flex customers can buy 24x7x365 support; small businesses can simply purchase a license. A great option for businesses that want a commercially released and supported Keycloak component.

Gluu Hosting

Gluu can deploy a Flex instance for your business on Google’s serverless infrastructure. Software and configuration changes are managed via Gitops, and rolled out with zero downtime. Gluu’s Solo hosting offering is useful for businesses that want to use Flex, but do not want to worry about the care and feeding of another mission critical database and web infrastructure.

Open Source

Need FREE? Our software is based on the Linux Foundation Janssen Project. The free open source Janssen distribution (or “Jans” for short) uses the same core components as Flex, and you can dive in without even obtaining a Flex trial license. Visit our Github Project Home, or jump to the installation section of the Documentation.

Open Standards

Gluu Flex is a software distribution that enables your business to use open source infrastructure to deploy four critical identity APIs: OpenID, SAML, FIDO and OAuth. Value is created by enabling your business to securely connect the maximum number of applications, whether they are web, mobile or even command line.

Browse Docs Book Demo
OpenID

OpenID is a popular federated identity standard for web-based applications. Gluu and Janssen have accumulated many Open Provider Certifications. Not only can Gluu Flex act as an identity provider, but using Gluu’s OpenID Agama Project, you can link a person’s account with an external OpenID Provider.

SAML

SAML, the XML federation ancestor of OpenID Connect, is used primiarily by SaaS and older web-based enterprise applications for SSO. In Flex, the Keycloak component provides the SAML IDP endpoints and acts as a SAML SP via the federation broker (to utilize an external SAML IDP).

FIDO

A set of standards that enable the hardware, operating system, browser, and identity provider to authenticate a person using phishing-resistent, client-side biometrics.  Flex provides FIDO endpoint that support  USB, Bluetooth. and platform authenticators.

OAuth

OAuth is used to authenticate a person using a mobile device, or to authenticate software–identity is not just for humans! Using Flex, your business can manage how software clients register an asymetric secret, and which API endpoints they are authorized to access.

Super Gluu

Passkey

Apple, Google and Microsoft have collaborated on federating FIDO capabilities they expose through their operating systems. This cooperation enables businesses to rollout passwordless authentication with unparalleled usability, security and deployability.

One Time Password

Gluu has OOTB support for TOTP, which is the standard behind Google Authenticator and other one-time password applications. Not a panacea, but not a bad option for some use cases.

SMS

While control of an SMS number is not a great indicator of human identity, it may the best or only option for your business. Don’t worry–we have your covered. Sign up with Twilio, and send SMS OTP codes in seconds.
